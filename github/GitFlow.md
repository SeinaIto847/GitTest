## git cloneする
1. ローカルで管理するディレクトリを作成する
   ```sh
   cd Document/study
   mkdir git
   cd git
   ```
2. リモートのリポジトリをクローンする
   ```sh
   git clone https://github.com/SeinaIto847/GitTest.git
   ```
